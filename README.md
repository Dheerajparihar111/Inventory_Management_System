# 📦 Inventory Management System 

A professional **Command Line Interface (CLI) Inventory Management System** built using **Java** with Object-Oriented Programming concepts.
This project allows users to manage products with **name, quantity, price, and total value**.

The system supports full inventory operations like add, view, search, update, and remove items.

---

## 🚀 Features

✔️ Add Item
✔️ View Items
✔️ Search Item
✔️ Update Item
✔️ Remove Item
✔️ Price & Quantity management
✔️ Total value calculation
✔️ Menu-driven CLI interface
✔️ Simple OOP design using class & objects

---

* 💻 Command Line Interface (CLI)

## 📌 Product Details Stored

Each product contains:

* Name
* Quantity
* Price
* Total Value

Total Value Formula:

```
Total = Quantity × Price
```
